# Punch Clock Mini Program

A WeChat Mini Program for daily clock-in tracking with duration statistics, weekly/monthly summaries, and cloud-based data sync.

## Features

- **One-tap Clock In** — Large circular button for quick daily check-ins
- **Duration Tracking** — Automatically calculates daily duration (precise to seconds) from earliest to latest clock-in
- **Date Navigation** — Browse historical records by switching dates with arrow buttons
- **Weekly & Monthly Statistics** — View total days, total duration, and average daily duration
- **Daily Breakdown** — Detailed per-day duration list within each week/month
- **Cloud Sync** — Data synced via WeChat Cloud Development, accessible across devices

## Tech Stack

- WeChat Mini Program (WXML, WXSS, JS)
- WeChat Cloud Development (Cloud Database)

## Setup

1. Open the project in **WeChat DevTools**
2. Replace `your-appid` in `project.config.json` with your Mini Program AppID
3. Enable **Cloud Development** in WeChat DevTools
4. Replace `your-cloud-env-id` in `app.js` with your cloud environment ID
5. Create a database collection named `daily_records` in the Cloud Console
6. Set collection permissions to **"Creator read/write only"**
